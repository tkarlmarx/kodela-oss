# Contributing to Kodela

Thanks for your interest! Kodela Community Edition is AGPL-3.0.

## Development
- Node 24+, pnpm 10+. `pnpm install` at the root.
- `pnpm typecheck` — whole-workspace typecheck.
- `pnpm test` — run tests.
- New source files should carry an SPDX header (`// SPDX-License-Identifier: AGPL-3.0-only`).

## Pull requests
- Keep changes focused; include tests.
- Describe the *why* — Kodela is a system of record for why code changes.
- Sign off your commits (DCO): `git commit -s`.

## Scope
This repository is the **Community Edition**. Commercial/Cloud/Enterprise features
live in a separate repository and are out of scope here.
