# Roadmap

Community Edition direction (subject to change):

- **Plugin SDK** — stable extension points for storage adapters, LLM providers, and IDE integrations.
- **Fused memory graph** — deepen the code-structure ⇄ decision-graph fusion and visualization.
- **More AI-tool integrations** — beyond Claude Code / Cursor / VS Code.
- **Docs & examples** — quick starts, sample plugins, architecture deep-dives.

Have an idea? Open a Discussion or a feature request.
